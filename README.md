## git using method
### 克隆
git clone https://github.com/ripplesaround/chat-system.git
### 上传
当一版比较完善的修改完成时（最好木有bug）：
1. git pull origin master
2. git add .
3. git commit -m '这里可以写修改了哪些内容'
4. git push origin master
### 更新master代码到本地
git pull origin master
